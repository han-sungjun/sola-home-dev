# 더운정픽 UI 블루 프리미엄 3차 보정본

## 주요 보정
1. 회원 탈퇴 영역의 레드 톤을 블루/슬레이트 계열의 차분한 주의 UI로 보정했습니다.
2. 공개앱 GNB와 관리자 GNB에 남아 있던 이모지형 아이콘을 SVG/라인 아이콘 중심으로 교체했습니다.
3. GNB 내부 스크롤 시 하단 영역이 비쳐 보이는 현상을 줄이기 위해 GNB 시트 높이, 내부 스크롤, overscroll 동작, 하단 여백을 보정했습니다.
4. 로그인/회원가입/앱/관리자 로딩을 브랜드 심볼 기반 블루 프리미엄 로딩 오버레이로 보정했습니다.
5. 브랜드 표현은 텍스트형 로고보다 `brand-symbol.svg` 심볼 마크 중심으로 통일했습니다.

## 확인 순서
- 공개앱 GNB → 설정/계정관리/회원탈퇴 영역
- 공개앱 GNB 하단 끝까지 스크롤
- 관리자 GNB → 전체 기능/하단 네비 겹침 확인
- 로그인/회원가입/앱 이동 시 로딩 오버레이 확인
- 브랜드 블록과 GNB 우측 심볼 마크 품질 확인

## 주의
기존 기능 로직은 유지하고 CSS/SVG/일부 마크업 위주로 보정했습니다. 개발계 확인 후 화면별 미세 여백과 실제 데이터 기반 아이콘 매핑은 추가 보정할 수 있습니다.
