더운정픽 du-layer 통합 3차 변경사항

1. 동적 이미지 팝업까지 du-layer 규격 적용
- 소식 이미지 팝업: du-layer / du-layer--modal / du-layer__panel / du-layer__header / du-layer__body 적용
- 혜택 상세 이미지 확대 팝업: du-layer / du-layer--fullscreen / du-layer__panel / du-layer__header / du-layer__body 적용

2. 헤더 통합 3차
- 소식 이미지, 혜택 이미지 팝업 헤더를 du-layer__header-main / du-layer__header-actions 구조로 정리
- 소식 이미지와 혜택 이미지는 헤더 구분선 유지
- AI 이미지 확대는 헤더 구분선 제거 유지

3. 닫힘 정책 정리
- 소식 이미지 팝업 바깥 클릭 닫힘 제거
- 혜택 이미지 팝업 바깥 클릭 닫힘 방어 유지
- 닫기는 X 버튼 중심으로 유지

4. 혜택 상세 이미지 확대 팝업 보정
- 이미지 비율에 따라 팝업 높이가 변하지 않도록 고정 높이 유지
- 스와이프 영역 흔들림 방지
- 흰 여백은 반투명 유리 느낌 유지
