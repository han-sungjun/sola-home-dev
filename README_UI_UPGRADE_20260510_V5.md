# 더운정픽 UI 고급화 v5

## 반영 내용

1. 심볼 마크 크기 보정
- 기존 `brand-symbol-real.png`가 실제 심볼이 너무 작게 들어가던 문제를 수정했습니다.
- 실제 U + 체크 심볼이 박스 안에서 크게 보이도록 재생성했습니다.

2. 로딩 오버레이 재정리
- 대각선/핑크/하단바 느낌이 남던 로딩 표현을 제거했습니다.
- 중앙 심볼 + 블루 링 방식으로 다시 고정했습니다.

3. 관리자 GNB 어두운 잠김 현상 보정
- GNB overlay가 시트 위로 올라와 화면이 잠긴 것처럼 보이는 문제를 방지하기 위해 z-index와 배경을 재정리했습니다.
- GNB sheet가 항상 overlay 위에 오도록 수정했습니다.

4. GNB 스크롤/하단 비침 재보정
- GNB body 내부만 스크롤되도록 고정했습니다.
- GNB open 시 하단 네비는 숨김 유지합니다.

5. 잔여 이모지 느낌 완화
- GNB 및 일부 메뉴 영역에서 이모지성 표현을 SVG fallback 기준으로 보정했습니다.

## 다음 권장 단계
- 제안서 기준으로 상단 헤더 구조를 본격 개편하는 v6 작업을 진행하면 됩니다.
