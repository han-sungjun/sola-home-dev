# 더운정픽 UI 블루 프리미엄 2차 보정본

## 주요 보정
- 로그인 페이지 레드 CTA를 블루 프리미엄 CTA로 변경
- 공개앱/관리자 상단 브랜드 영역을 반투명 브랜드 블록 형태로 보정
- 작은 크기에서도 깨져 보이지 않도록 `brand-symbol.svg`를 단순 stroke 기반 심볼로 재작성
- 공개앱/관리자 GNB의 주요 이모지 버튼을 SVG 아이콘으로 교체
- 관리자 자동 로그아웃 UI는 숨김 유지, 기능 로직은 그대로 유지
- 하단 네비바 v1 스타일은 유지하되 전체 블루 톤과 맞도록 보정

## 확인 포인트
1. 로그인 화면의 `앱으로 빠르게 이용하기` 버튼이 블루 계열인지 확인
2. 공개앱/관리자 상단 브랜드 블록이 반투명 배지처럼 보이는지 확인
3. GNB 상단의 큰 로고가 깨지지 않는지 확인
4. 관리자 GNB/하단 네비에서 이모지 느낌이 줄었는지 확인
5. 기존 로그인/회원가입/혜택/공지/관리 기능 동작 여부 확인
