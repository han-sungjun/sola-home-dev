# 더운정픽 블루 프리미엄 UI 1차 적용본

## 적용 목적
개발계에서 제안 디자인을 빠르게 확인하기 위한 1차 UI 고급화 패치입니다. 기존 기능/데이터 로직은 최대한 유지하고 CSS와 SVG 아이콘 중심으로 적용했습니다.

## 주요 적용 내용

### 공개앱
- 상단 헤더/히어로 영역을 블루 프리미엄 톤으로 통일
- 브랜드 블록을 반투명 글래스 느낌으로 정리
- 하단 네비바를 네이비/블루 그라데이션 기반으로 고급화
- 선택된 하단 메뉴는 블루 글래스 캡슐 형태로 강조
- GNB 메뉴의 헤더/배너/카드/검색 영역을 블루 톤으로 정리
- 알럿/상세 모달에 블루 포인트 라인, 버튼, 아이콘 톤 적용

### 관리자 페이지
- 운영자 페이지 상단/히어로/하단 네비를 블루 프리미엄 톤으로 통일
- 관리자 GNB와 메뉴 카드 톤 정리
- 자동 로그아웃 기능은 유지하되 UI 타이머는 숨김 처리

### SVG 아이콘
- `/icons/internal/brand-symbol.svg` 신규 추가
- 주요 내부 SVG 아이콘을 블루/네이비/골드 톤으로 재정리
  - home, benefit, favorite, top5, all-menu, notice, calendar, map, ai-assistant, lock, user, settings 등

## 확인 포인트
1. 로그인/회원가입/공개앱/관리자 페이지에서 블루 톤 통일감 확인
2. 하단 네비 활성 상태가 과하지 않고 선명한지 확인
3. GNB 메뉴에서 브랜드 블록과 검색/서비스 카드 가독성 확인
4. 혜택 상세/공지 상세/알럿 모달의 여백과 버튼 강조 확인
5. 관리자 페이지 자동 로그아웃 기능은 유지되는지 확인

## 주의
- 1차 확인용 적용본입니다.
- 기존 단일 HTML 파일 구조를 유지하기 위해 CSS override 방식으로 적용했습니다.
- 화면 확인 후, 2차 작업에서는 중복 CSS 정리 및 컴포넌트 단위 정리 권장합니다.
