# 더운정픽 UI 블루 프리미엄 v4

## 1순위 보정 반영

1. 상단/잔여 로딩바 숨김
- `.top-loader`, `.top-progress`, `.loading-bar` 계열을 숨김 처리했습니다.
- 로딩은 중앙 브랜드 심볼 기반 오버레이로만 표시되도록 보정했습니다.

2. 실제 심볼마크 적용
- 디자인 가이드 이미지에서 추출한 `brand-symbol-real.png`를 추가했습니다.
- 기존 단순 SVG 참조를 실제 심볼 이미지 중심으로 교체했습니다.

3. GNB 스크롤 비침 보정
- GNB overlay/sheet를 `100dvh` 고정 구조로 정리했습니다.
- GNB 내부 body만 스크롤되도록 분리했습니다.
- GNB 오픈 시 하단 네비를 숨겨 하단 배경 비침을 줄였습니다.

4. GNB 이모지 잔존 제거
- 정적 이모지 문자열 일부를 제거했습니다.
- 아이콘 영역은 SVG/이미지 아이콘만 보이도록 CSS guard를 추가했습니다.

5. 회원 탈퇴 레드톤 제거
- 회원 탈퇴 버튼/안내/태그를 레드 대신 슬레이트·블루 계열 주의 UI로 조정했습니다.

## 확인 순서
1. 로그인 / 회원가입 로딩
2. 공개앱 브랜드 심볼
3. 공개앱 GNB 스크롤 끝부분
4. 회원 탈퇴 안내 영역
5. 관리자 GNB / 관리자 로딩
